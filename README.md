# Нейроассистенты для автокомпании

## Описание

Учебный проект: три нейроассистента для вымышленного автодилера (продажи, внутренний HR, поддержка клиентов после покупки). В репозитории — системные промпты, базы знаний (FAQ) и скрипты диалогов; материалы можно перенести в GPTs, бота или CRM.

## Что реализовано

- 3 ассистента (Sales, HR, Support)
- системные промпты
- базы знаний
- скрипты общения

## Структура проекта

| Путь | Содержание |
|------|-------------|
| `final_project_doc.md` | Итоговый документ для сдачи и портфолио: компактный обзор решения; полные FAQ, скрипты и промпты — в папках выше |
| `assistants/sales_assistant.md` | Системный промпт Sales |
| `assistants/hr_assistant.md` | Системный промпт HR |
| `assistants/support_assistant.md` | Системный промпт Support |
| `knowledge_base/sales_faq.md` | FAQ отдела продаж |
| `knowledge_base/hr_faq.md` | FAQ для сотрудников (HR) |
| `knowledge_base/support_faq.md` | FAQ сервиса и поддержки |
| `scripts/sales_scripts.md` | Скрипты общения Sales |
| `scripts/hr_scripts.md` | Скрипты общения HR |
| `scripts/support_scripts.md` | Скрипты общения Support |

## Как использовать

- можно загрузить в GPTs (скопировать системный промпт из `assistants/*.md` и приложить FAQ к файлам знаний)
- можно использовать в Telegram-боте (промпт + RAG или вставка фрагментов FAQ)
- можно интегрировать в CRM (контекст из карточки клиента, эскалация на оператора)
